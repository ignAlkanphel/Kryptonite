# Kryptonite

---
A library for [Palladium](https://git.threetag.net/ThreeTAG/Palladium) that adds a ton of new abilities, conditions, actions, and no kryptonite! <br>

Notably adapts things from the [Apoli](https://github.com/apace100/apoli/releases) (licensed under MIT) mod for Palladium.
